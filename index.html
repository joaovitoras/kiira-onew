Bem Vindo ao projeto Kiira Onew

Este app é um portfólio para Designers...

Para obter esse app basta utilizar o git clone

$ git clone git://github.com/joaovitoras/kiira-onew.git

Antes de iniciar o app rode

$ cd kiira-onew
$ rake db:setup
$ rake db:seed

Agora rode o servidor rails

$ rails s

Upload de Imagens feitos com a gem carrierwave

Em development configure o local para o upload de suas imagens no arquivo image_uploader.rb

 def store_dir
    "public/#{model.class.to_s.underscore}/#{mounted_as}/#{model.id}"
 end

Em produção é utilizado o dropbox para servidor de imagens com a gem carrierwave-dropbox

Configure o as variaveis de ambiente para acesso a sua conta no dropbox dropbox.rb.

 CarrierWave.configure do |config|
  config.dropbox_app_key = ENV["APP_KEY"]
  config.dropbox_app_secret = ENV["APP_SECRET"]
  config.dropbox_access_token = ENV["ACCESS_TOKEN"]
  config.dropbox_access_token_secret = ENV["ACCESS_TOKEN_SECRET"]
  config.dropbox_user_id = ENV["USER_ID"]
  config.dropbox_access_type = "dropbox"
end
